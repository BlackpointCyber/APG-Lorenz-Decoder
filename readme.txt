Lorenz_Decoder

A Python script to extract and decode Lorenz ransomware BitLocker recovery key and password from main.dll

usage: python decode.py main.dll

During the analysis of a recent attack conducted by the Lorenz ransomware group, Blackpoint Cyber's Adversary Pursuit Group (APG) found the BitLocker recovery key and password encoded within main.dll. 

This script has only been tested on the sample obtained during this incident.

For additional information, check out our blog. https://blackpointcyber.com/resources/blog/the-chaos-of-lorenz/
